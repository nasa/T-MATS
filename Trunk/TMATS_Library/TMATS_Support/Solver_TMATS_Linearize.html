T-MATS: Linearization Library Block
Determine State-space equation for a given system where:

(xdot - xdot_0) = A*(x - x0) + B*(u - u0)
(y - y0) = C*(x - x0) + D*(u - u0)

During operation each plant state (x) and input (u) will be set to an initial condition then perturbed separately. Perturbed inputs and system effects are compared against the baseline to determine the A,B,C, and D matricies. Perturbation sized are specified below. x0, u0, xdot_0, and y0 values are initial conditions. Matrix sizes are specified as:
xdot and x : mx1
y : px1
u : nx1
A : mxm
B : mxn
C : pxm
D : pxn


Linearize Inputs:
Lin. InputDescription
x0value of x to be linearized around, initial x (mx1)
u0value of u to be linearized around, initial u (nx1)
xdotdelta x. Determined form x perturbation (mx1)
ysystem output data (px1)
EnableEnables the linearize block. If block is disabled x0 and u0 will be passed through to outputs x and u respectively
ResetReset the linearize block

Linearize Outputs:
Lin. OutputDescription
ABCD_matricesbus containing the ABCD matrices
xcurrent value of x, used for perturbing the system (mx1)
ucurrent value of u, used for perturbing the system (nx1)
xdot_0xdot value when x = x0 and y = y0 (mx1)
y0y value when x = x0 and y = y0 (px1)
Iternumber of time Linerize block has run
ActiveLinearize block is actively running
