T-MATS: MATLAB functions

The T-MATS package contains MATLAB functions for doing certain Thermodynamic operations. Information about these sets of functions are located in the users manual.

As a quick reference the T-MATS functions are also located below. They can be accessed by using the class structure TMATS.* .

Units consist of:
temperature - R
pressure - psia
enthalpy = BTU/lbm
entropy - BTU/(lbm*R)
FAR - NA

TMATS.* functions
Class MethodDescription
[T] = h2t(h)Look up temperature(T) based on enthalpy(h).
[h] = t2h(T)Look up enthalpy(h) based on temperature(T)
[s] = PT2s(T,P)Look up entropy(s) based on pressure(P) and temperature(T)
[T] = sP2T(s,P)Look up temperature(T) based on entropy(s) and pressure(P)
PlotCMap(NcV,WcA,PRA,EffA)Plot compressor map based on an Nc vector, Wc array, PR array, and Eff array.
addtional inputs can be included by with the format PlotCMap(_,Name,Value,...,Value)
type "help TMATS.PlotCMap" for acceptable inputs and format.
An exmaple of this command might look like:
TMATS.PlotCMap(NcV,WcA,PRA,EffA,'scalar',s_Wc,s_PR,s_Eff,'effcolor',0.5,'slformat','%2.2f','offset',0.03,'defsurge',WcA(:,1), PRA(:,1));
PlotTMap(NcV,PRV,WcA,EffA)Plot turbine map based on an Nc vector, PR vector, Wc array, and Eff array
addtional inputs can be included by with the format PlotTMap(_,Name,Value,...,Value)
type "help TMATS.PlotTMap" for acceptable inputs and format.
An exmaple of this command might look like:
TMATS.PlotTMap(NcV,WcA,PRA,EffA,'scalar',s_Wc,s_PR,s_Eff,'effcolor',0.5,'slformat','%2.2f','offset',0.03);
