T-MATS: Data Output Library Block
This Block saves the input variable to a file at each time step. Note, this will overwrite the previous value.

Data Output Inputs:
Duct InputDescription
VarVariable to be saved

Data Output Mask Variables:
Data Output Mask VariableDescription
FVar_MFile name for variable output
Var_MVariable name that will appear in the file
SVar_MVariable surname that will appear in the file
Note: "/" or "\" will be replaced by "_" when generating the name
